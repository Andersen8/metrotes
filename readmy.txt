Использующиеся технологии:
- сборка проекта Maven
- веб контейнер Tomcat 8
- версия java 8
- Spring mvc, context
- реляционная БД PostgresSQL

Настройки к подключению БД:
- имя базы: metro полный путь: jdbc:postgresql://localhost:5432/metro
- пользователь: postgres
- пароль: password

Для сохранения файлов использовался абсолютный путь к директории проекта,
который при импорте нужно будет редактировать (CSV_Util.ABSOLUTE_PATH).
